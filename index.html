Hello! I am Manu, a biologist turned developer. I am part of the Bähler lab in University College London.

I am currently working on ShareYourCloning, a project to develop:

  • A data model to represent cloning strategies
  • A web interace to plan your cloning and export it in that format

If you are interested, visit the ShareYourCloning repository.

All this is part of a bigger project, Genestorian, which aims to create an Open Source web application to manage laboratory collections of plasmids, strains and cell lines. You can find a good introduction to this project in my last conference poster
